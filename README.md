#  Minecraft Hypixel Trading Model

An algorithmic trading bot designed to analyze the Hypixel SkyBlock economy. This project scrapes real-time market data, performs feature engineering, and utilizes Machine Learning models to predict price trends for profitable trades.

##  Key Features
* **Real-Time Data Collection:** Automated web scraping bot running 24/7 on AWS EC2 (m7i-flex.large).
* **Cloud Integration:** Optimized for Linux (Ubuntu) cloud environments with headless Selenium.
* **Data Pipeline:** Continuous data logging to CSV/Cloud Storage for time-series analysis.
* **Machine Learning (In Progress):** Price prediction using Reinforcement Learning and LSTM (Long Short-Term Memory) models.

##  Tech Stack
* **Language:** Python 3.x
* **Cloud:** AWS EC2 (Ubuntu 24.04 LTS)
* **Libraries:** Selenium, Pandas, NumPy, Scikit-learn
* **Tools:** Git, Vim, Screen/Tmux
